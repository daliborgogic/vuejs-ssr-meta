# Vuejs Server Side Rendering Meta

[Live Demo](https://vuejs-ssr-meta-tmflbxpghm.now.sh/)

## Build Setup

**Requires Node.js 6+**

``` bash
# install dependencies
npm i

# serve in dev mode, with hot reload at localhost:8080
npm run dev

# build for production
npm run build

# serve in production mode
npm start
```
